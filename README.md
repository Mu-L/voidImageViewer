# void Image Viewer
A lightweight image viewer for Windows with GIF/WEBP support.  
Opens and displays BMP, GIF, ICO, BMP, JPG, TIF and WEBP images as fast as possible.  
Animate GIF/WEBP files as accurately as possible.
